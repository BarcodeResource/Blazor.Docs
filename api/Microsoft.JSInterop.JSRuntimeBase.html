UntrackObjectRef(DotNetObjectRef)

Stops tracking the .NET object represented by the DotNetObjectRef. This allows it to be garbage collected (if nothing else holds a reference to it) and means the JS-side code can no longer invoke methods on the instance or pass it as an argument to subsequent calls.
